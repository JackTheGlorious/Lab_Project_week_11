Welcome to Jon's Moving Planner. This is a web app designed to help you plan your next move. Using our tools you can learn everything you need to know about moving and then use our planner tools to make a plan within our app to plan your move.
To start learning about what do when you move select Planning intructions. To start planning your move select Planner. Select about to learn more about us. Select contact to reach our for any questions or inquiries.

Planning Application

The planning application will help you shedule your next move. You can create an itemized plan to make your move as easy as possible. The planner will take your list of items and help put it to a time schedule.

Usage Instructions

To begin using the planner select the Planner button on the navigation bar. It is recomended you first use our learning resources before planning. Once in the planner select the Add Item page in the navigation bar to start adding yout things to do. From there you can select Display Items on the nav bar to edit and delete items from your planner.
